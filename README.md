# The Collegium of Minds - Static Site

This is a static version of "The Collegium of Minds" website, designed to be hosted on GitHub Pages or any static web server.

## Features
- Periodic table aesthetic
- Dark mode support (persisted in localStorage)
- Responsive grid layout
- Atmospheric background animations
- Google Fonts (JetBrains Mono & Merriweather)
- Tailwind CSS via CDN

## Structure
- `index.html`: Main structure and content
- `style.css`: Custom styles and animations
- `script.js`: Theme toggle and interactivity
- `assets/`: Image assets (Logo)

## How to use
Simply open `index.html` in any web browser.
